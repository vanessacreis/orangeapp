## 🏁 Iniciando o projeto

- Clone o repositório

```
https://github.com/vanessacreis/orangeapp.git
```

- Acesse a pasta criada

```
cd orangeapp
```

- Instale os pacotes necessários

```
npm install
```

- Inicie o projeto

```
npm start
```

- A aplicação abre por padrão na porta 3000

```
Acesse http://localhost:3000/
```
